# Client 3D Controller
---

All of the sources (.h and .cpp files) of this project are described in the docs folders.
This project was made for testing commands that later will work on the ESP32.

## Requisites:
- Qt: you can see the CmakeLists.txt folder for more information.
If you want to change the 3D model you will need the balsam app, is inside the sources folders of the Qt installation: in my case(C:\Qt\6.9.2\mingw_64\bin\balsam.exe).
